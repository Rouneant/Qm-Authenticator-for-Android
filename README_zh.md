### 特性

- **密钥安全**：所有密钥**完全离线**存储在设备本地
- **开源透明**：完整的源代码开放，安全性和隐私性可供任何人审查。
- **标准兼容**：支持基于时间的一次性密码算法（TOTP），与所有使用此标准（如 Google, GitHub, Microsoft 等）的服务兼容。
- **简洁易用**：**Material Design 3** 设计风格，提供直观、流畅的用户体验。
